# Segmentation
Image segmentation challenge for Digital Image Processing @PKU, 2021.

File layout:
```
| - src
    | - augment.py
    | - model.py
    | - train.py
    ...
| - data
    | - complex
        | - train
            | - (images)
        | - val
            | - (images)
        | - label
            | - (labels whose filenames are the same as corresponding images)
    | - simple
        | - train
        | - val
        | - label
```
